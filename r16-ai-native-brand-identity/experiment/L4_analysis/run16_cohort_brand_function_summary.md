## Hypothesis Results

### H1: Cohort x Condition Interaction — SUPPORTED

F = 26.81, p < .001, partial eta-sq = .335.

Brand Function specification (Condition B) affects DCI differently across cohorts. The interaction effect is large (eta-sq > .04 criterion easily exceeded).

DCI reduction from Baseline (A) to Structural (B):

| Cohort | Implied Dims | A->B Reduction |
|--------|-------------|----------------|
| C2 (Taste Curator) | Semiotic, Social | +3.20 pct pts |
| C1 (Green Advocate) | Ideological, Narrative | +1.65 pct pts |
| C7 (Experience Collector) | Experiential, Narrative | -1.53 pct pts |
| C5 (Long Habit) | Temporal, Cultural | -2.97 pct pts |
| C3 (Spreadsheet Shopper) | Economic, Experiential | -3.02 pct pts |

Brand Function increases DCI for cohorts whose implied dims are "soft" (C1, C2) but decreases DCI for cohorts whose implied dims include Economic (C3, C5). When the Brand Function reveals the full 8-dimensional specification, Economic-focused observers redistribute weight away from their dominant dimension.

### H2: DCI Reduction Range — NOT SUPPORTED

Range: 6.22 percentage points (criterion: 15). The interaction is statistically significant but the magnitude of differential DCI reduction across cohorts is smaller than hypothesized. This suggests Brand Function operates more uniformly across observer types than predicted.

### H3: Run 14 Null Replication — CONFIRMED

No cohort shows significant C vs B difference. All p > .12.

The Run 14 null finding generalizes across cohorts: detailed per-dimension enrichment produces no incremental benefit beyond structural completeness. This holds for all 5 tested cohort types, from Green Advocates to Spreadsheet Shoppers.

## Key Findings

1. **Brand Function x Cohort interaction is real** (eta-sq = .335): how observers respond to Brand Function depends on their behavioral profile.
2. **Economic-focused cohorts reduce DCI with Brand Function**: when given full dimensional data, price-focused observers diversify their weighting. Brand Function corrects for Economic Default more in these observers.
3. **"Structural, not granular" generalizes**: Run 14's null finding (enriched detail adds nothing beyond structure) holds across all 5 cohort types.
4. **DCI range is narrower than expected**: the differential effect across cohorts is significant but modest (~6 pct pts), suggesting Brand Function is more of a universal correction than a cohort-specific intervention.
