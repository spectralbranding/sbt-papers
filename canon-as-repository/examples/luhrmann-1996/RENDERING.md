# Rendering: Baz Luhrmann's Romeo + Juliet (1996)

## Rendering Type
Rendering — the spec is preserved; the medium and visual language are transformed.

| Parameter       | Value                                              |
|-----------------|----------------------------------------------------|
| Medium          | Film                                               |
| Director        | Baz Luhrmann                                       |
| Year            | 1996                                               |
| Style           | Postmodern baroque, MTV-generation visual language |
| Setting         | Verona Beach (modern coastal city)                 |
| Weapons         | Guns (branded as "Sword 9mm", "Dagger")            |

## Specification Gap Analysis

The film's central rendering decision is preserving Shakespeare's language in a modern visual
environment, creating productive tension between archaic speech and contemporary setting.
Characters speak in Elizabethan verse while driving cars and firing automatic weapons.
This is not contradiction — it is the rendering's argument: the story is not period-specific.
The language survives translation into any surface context.

The "guns branded as sword types" device is the clearest single rendering choice. It satisfies
the honor-code constraint (physical confrontation is required) while adapting to the medium
(film, 1996, action-genre conventions) and style (modern Verona Beach). The spec constraint
is preserved; the surface instantiation is transformed.

Mercutio (Harold Perrineau) is played with a level of theatrical excess — drag costume for
the party, heightened physicality — that amplifies the Queen Mab speech's requirements
while adding an additional register (performativity, concealment) not in the spec. This is
a rendering addition, not a spec violation.

The ending slightly overrides the tone constraint. The spec requires grief and quiet recognition;
the film's final news-broadcast framing and score add emotional uplift. This is a small
but documentable override.

## Rendering Assessment

This is a high-compliance rendering with one surface-level tonal override at the ending.
The spec's structural constraints are all met. The visual and stylistic transformation
demonstrates that the spec is robust to very large surface changes provided the structural
rules are followed.
