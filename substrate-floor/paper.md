# The Substrate Floor: Nested Noise Floors for Auditable Abstention Across Heterogeneous Instruments

Dmitry Zharnikov

ORCID: 0009-0000-6893-9231

DOI: [10.5281/zenodo.20840422](https://doi.org/10.5281/zenodo.20840422)

Working Paper v1.0.0 – June 2026

## Abstract

An instrument abstains honestly when its signal falls below a noise floor it computes for itself. This paper generalizes that discipline to an ensemble of heterogeneous instruments: a finding survives only if its signal clears the outermost noise floor in play, where floors nest — operator within artifact within substrate. The same move lifts up a level: a cross-instrument ensemble abstains below a *substrate floor*, the dispersion of instruments' verdicts on one aligned claim. Reconciliation returns a typed verdict — corroborated, contested, substrate-conditional, or jointly-unresolved — never a forced point estimate; agreement is triangulation, disagreement is the floor itself. A no-rescue lemma forbids agreement among individually-noisy instruments from manufacturing a finding. The rule composes instruments of different kinds that share no output space, reconciling an eight-dimensional measurement vector and a categorical specification audit on an aligned ordinal verdict, and below a specification instrument's coherence floor abstention is mandated by a coverage-impossibility result. The contribution is an open-source computational framework — already shipped, with worked cases reproducing all four verdicts and a seeded Monte-Carlo map of where it diverges from meta-analysis, sensor fusion, and selective prediction. It is a typed-verdict lattice over nested floors, not a relabeling of heterogeneity statistics.

**Keywords:** auditable abstention, noise floor, heterogeneous instruments, verdict reconciliation, uncertainty quantification, ontology alignment, selective prediction, reproducibility of results

---

Patterns in scientific data are increasingly detected not by one instrument but by an ensemble of heterogeneous ones — deep-learning embeddings, large-language-model audits, specification checklists, human-coded content analyses, public benchmark leaderboards — each carrying its own uncertainty. Consider a concrete and now-routine case: three public benchmarks rank the same set of language models, and the rankings disagree; one benchmark calls model A the strongest, another calls the contest a tie, and a downstream pipeline must report a single ordering. When such verdicts conflict, the standard resolutions reintroduce a hidden oracle. Averaging the instruments' outputs presumes a common scale and a meaningful mean; taking the most confident instrument presumes confidence tracks correctness; majority voting presumes independence and individual calibration. Each move converts a legitimate *cannot tell* into a point estimate whose provenance cannot be audited. The reproducibility literature has documented how often such hidden degrees of freedom produce findings that do not replicate [@baker-2016-1500-scientists-reproducibility; @ioannidis-2005-most-published-findings-false; @munafo-2017-manifesto-reproducible-science], a failure mode that in machine-learning-based science frequently traces to exactly this kind of unaudited methodological choice [@kapoor-2023-leakage-reproducibility-crisis] and that national science-policy bodies now treat as a systemic threat to reliable inference [@national-academies-2019-reproducibility-replicability]. The difficulty is sharpest when the instruments are genuinely heterogeneous: when one returns a numeric vector and another a categorical audit, when they were built by different parties on different vocabularies, and when none can be assumed an unbiased rater.

The stance taken here is that an instrument can be trusted to abstain only against a noise floor it computes for itself, and the question is how that discipline extends to an ensemble. The within-instrument case is one published instance: a measurement instrument that reconstructs cohort-resolved perception vectors from public artifacts reports every comparison relative to two noise floors it computes — an operator-variance floor across model pairs and a source-sensitivity floor — so each comparison is a falsifiable signal-to-noise test that resolves or abstains [@zharnikov-2026ax-brand-spectrometer]. The contribution of this paper is the next level up: a general computational primitive for any multi-instrument pipeline that answers, when several instruments — possibly of different kinds — are composed on one claim, what the floor is and what clearing or failing it returns.

The paper makes three contributions. First, it states a *nested noise-floor rule* (the *nested-floor rule*): floors nest operator within artifact within substrate, a finding survives only if it clears the outermost floor in play, and a no-rescue result establishes that agreement among individually-noisy instruments cannot lift a consensus above that floor. Second, it specifies a typed-verdict reconciliation calculus (the *reconciliation lattice*) — returning corroborated, contested, substrate-conditional, or jointly-unresolved — computed over a cross-instrument dispersion floor (the *substrate floor*) on a common normalized verdict axis, with two mandatory gates: ontology alignment as a step-0 (apparent conflict is often vocabulary mismatch) and self-certification (an instrument that cannot vouch for its own calibration is excluded, never averaged). Third, it demonstrates *cross-kind abstention*: a measurement noise floor and a specification coherence floor are two instances of one discipline, reconciled on the aligned claim's verdict, with abstention below the coherence floor mandated rather than conventional. The framework — a reproducible, instrument-general typed-verdict calculus with an open-source reference implementation — is the contribution, not a new heterogeneity statistic; the relation-to-prior-art analysis in a later section shows it returns verdicts that meta-analysis, sensor fusion, metrology uncertainty budgets, and selective prediction, by construction, do not.

The practical stake is reproducible, auditable pattern detection across heterogeneous data sources. When an automated pipeline composes embeddings, model audits, and benchmark scores into a reported pattern, the question of which findings are real and which are artifacts of an unaudited reconciliation step is exactly the reproducibility question the science-policy literature raises [@munafo-2017-manifesto-reproducible-science; @national-academies-2019-reproducibility-replicability]. The typed verdict makes that step auditable: every reported pattern carries, by construction, a provenance trail recording whether it was corroborated across instruments, contested between them, conditional on a single apparatus, or jointly unresolved — and the framework declines, rather than fabricates, a pattern when the evidence falls below the floor. Reproducibility, not a new statistic, is the value proposition.

## The Admissible Instrument

***What counts, and what does not***

The claim that one rule unifies abstention across instrument kinds is falsifiable only once the rule states what counts as a legitimate instrument. Three positive criteria define admissibility, and two explicit non-applications mark the boundary; the rule does not apply outside them.

**Definition 1 (admissible instrument).** An instrument is admissible only if it satisfies all three of the following. *Alignability*: its output for a shared claim can be brought, via an explicit alignment predicate (an exact, close, or narrow match in the sense of a standard ontological-mapping vocabulary [@matentzoglu-2022-sssom]), onto a common normalized verdict axis in the unit interval; reconciliation runs on that aligned verdict, never on the raw output. *Individually-computable floor*: the instrument must expose its own noise floor for the claim, computed without reference to the other instruments — a measurement instrument exposes an operator/artifact dispersion floor read live from its atlas, a specification instrument exposes a coherence floor read live from its audit. An instrument that can emit a value but cannot state the floor below which that value is noise lies outside the framework's scope, since the nesting rule has nothing to nest. *Self-certification*: the instrument must carry an auditable self-certification, so an integrator can exclude an instrument that cannot vouch for its own calibration; a black-box instrument with no audit log may be reported but never averaged or voted into a verdict.

Two non-applications fix the boundary. *Incommensurable subjective judgment*: instruments whose outputs are irreducibly subjective aesthetic or normative verdicts with no defensible common axis — whether an artwork is beautiful, an expert's taste — violate alignability; there is no honest normalized verdict to disperse, and forcing such outputs onto a shared axis would manufacture the very false precision the framework exists to refuse. *Floorless black-box models*: a model that emits a point prediction with no individually-computable floor and no self-certification is out of scope, because the rule cannot decide whether its signal clears a floor it never declared. This second non-application is also exactly the boundary that separates the substrate floor from naïve ensemble-disagreement scoring, which reads confident agreement among uncalibrated black boxes as evidence.

***Assumptions and their relaxations***

The framework rests on four assumptions, each relaxable with traced consequences. The aligned verdict axis is treated as a metric space on which an instrument's own floor and the distance between two instruments' verdicts are comparable quantities; under a purely ordinal axis the dispersion becomes a rank-disagreement statistic and the thresholds are recomputed on ranks, leaving the four verdict types and the nesting rule unchanged. Each instrument's declared floor is assumed an honest upper bound on its own noise, enforced operationally by reading floors live from committed artifacts rather than trusting hand-set values; an instrument that under-reports its floor is overridden by the live read, and one with neither a live artifact nor an honest declaration is inadmissible. Alignment predicates between each instrument's claim and the shared claim are assumed available upstream, produced by a federated negotiator [@zharnikov-2026at-negotiating-vocabularies-link]; absent an explicit predicate the verdict is treated as a close match at best and flagged as an alignment risk that can only lower confidence, never raise it. Finally, admitted instruments are assumed conditionally independent given the truth of the claim to the degree their floors claim — the honest weakening of the strong-independence assumption that fusion methods make; under correlated instrument error the dispersion under-states disagreement, the no-rescue lemma still bounds rescue from above, and corroborated verdicts are downgraded, a case examined in the robustness analysis.

## The Nested-Floor Rule

***Floors that nest***

Within one measurement instrument, two floors already nest. The operator floor is the dispersion of an instrument's reading across alternate operator pairs applied to the same artifacts; the artifact floor is its dispersion under leave-one-out resampling of the artifacts. A reading survives only if it clears the larger of the two — the instrument's combined floor. The substrate floor lifts this move one level up. Given several instruments resolving a shared claim, the *substrate dispersion* is the maximum pairwise distance over their verdicts on the common normalized axis. Floors then nest operator within artifact within substrate, and a finding survives only if its consensus signal clears the *effective floor*, the outermost in play:

$$\mathrm{effective\_floor} = \max\!\left(D,\; f_1,\; \dots,\; f_n\right),$$

where $D$ is the substrate dispersion and $f_i$ is each resolving instrument's own combined floor. The same single criterion — clear the outermost floor — composes within an instrument (operator to artifact) and across instruments (artifact to substrate), with no new mechanism introduced at the cross-instrument step. The signal-to-noise of the ensemble's consensus is $|c| / \mathrm{effective\_floor}$, banded by pre-registered resolve and marginal multipliers exactly as a single instrument bands its own readings.

***No rescue from agreement***

The nesting has an immediate and consequential corollary. Because the effective floor is a maximum that already includes every endpoint's own floor, it is at least as large as the floor of the noisiest resolving instrument. Hence if any participating instrument's signal lies below its own combined floor for the claim, the consensus cannot clear the effective floor, and no degree of cross-instrument agreement can lift it there.

**Proposition 1 (no rescue).** Let the consensus signal be $c$ and the effective floor be $\mathrm{effective\_floor} = \max(D, f_1, \dots, f_n)$. The consensus clears the floor if and only if $|c| > \mathrm{effective\_floor}$, and since $\mathrm{effective\_floor} \ge f_i$ for every $i$ by definition of the maximum, $|c| \le f_i$ for some $i$ implies $|c| \le \mathrm{effective\_floor}$. Therefore if any endpoint's signal lies below its own floor, the consensus cannot clear, regardless of how closely the instruments agree. The full statement and proof are given in Appendix A.

Proposition 1 states formally that cross-instrument agreement counts as triangulation only when each instrument's reading is itself above its own floor. Two instruments that agree closely while each reads below its own floor are agreeing on noise; their concord is not evidence. This is the single robustness criterion the rest of the paper builds on, and the point at which the framework departs from any method that treats inter-source agreement as confidence. The claim is falsifiable: a construction in which agreement between two instruments, each individually below its own floor, legitimately raised confidence above the effective floor would refute it.

## The Reconciliation Lattice

***The four typed verdicts***

Clearing or failing the effective floor is not a binary outcome but one of four typed verdicts, returned on the aligned verdict axis and never collapsed to a pooled point estimate. Let the resolving instruments be those whose own signal clears their own floor; the rest abstain. The verdict is a function of the resolver count, the substrate dispersion $D$, each endpoint floor, the consensus magnitude, the alignment predicate, and each instrument's self-certification.

*Corroborated* — at least two instruments resolve, agree within their floors ($D$ no greater than the largest endpoint floor), and the consensus clears the effective floor. This is cross-instrument replication, the highest-confidence verdict. *Contested* — at least two resolve but disagree beyond their floors ($D$ exceeds the largest endpoint floor). This is genuine conflict; by construction it escalates to human judgment rather than being averaged away. *Substrate-conditional* — exactly one instrument resolves above its floor while the rest abstain; the finding holds only under that apparatus, and the condition is reported. *Jointly-unresolved* — all instruments abstain, or the resolvers agree but on a consensus below the effective floor (agreement on noise, by the no-rescue lemma). The four regions partition the plane of dispersion against consensus-versus-floor; a categorical layer reports the normalized entropy of the resolve/abstain split, zero when the instruments are unanimous and one when maximally divided.

***Two gates: alignment and self-certification***

Two gates precede the lattice, each closing a failure mode that disagreement-scoring alone cannot see. The first is alignment as a mandatory step-0. Two instruments may not mean the same thing by the same term; apparent conflict is often a vocabulary mismatch (false conflict) and apparent agreement a shared word with divergent meaning (false agreement). Each verdict's alignment predicate to the shared claim is either declared by the instrument or computed from a live terms graph; a non-exact predicate is surfaced as a false-(dis)agreement risk, never silently trusted, and an alignment that cannot be established at all among the resolvers downgrades a would-be corroborated verdict to contested — one cannot corroborate across concepts a graph does not map to each other. This step operationalizes a vocabulary negotiation already specified for the corpus [@zharnikov-2026at-negotiating-vocabularies-link] and rests on a standard, typed mapping vocabulary [@matentzoglu-2022-sssom]. The second gate is self-certification: an instrument whose audit fails is excluded from the resolver set entirely, never down-weighted into the consensus. An unvetted instrument cannot pull the verdict, even toward the right answer. The verdict computation, including both gates, is given as Algorithm 1 in Appendix A; its correctness conditions are falsifiable — returning corroborated for two instruments in genuine conflict beyond their floors, or contested for an apparent conflict that exact alignment dissolves, would expose a defect.

## One Rule, Two Instrument Kinds

***The measurement floor and the coherence floor***

The unification claim is that the rule holds across instrument kinds that share no output space. The two instrument kinds and the primitives they expose are specified in companion methodological papers, each archived and reproducible; this paper reuses those primitives and contributes the cross-instrument calculus that composes them, which is established here by the open-source implementation, the worked cases, the public-benchmark reconciliation, and the Monte-Carlo characterization rather than by the companion papers. A measurement instrument's floor is an operator/artifact dispersion over model pairs and artifact draws [@zharnikov-2026ax-brand-spectrometer]; a specification instrument's floor is a *coherence floor* (a single scalar reading how internally consistent a specification is and how much of it the audit covered), computed as specification incoherence (one minus a specification-coherence index [@zharnikov-2026an-specification-readiness-empirical]) plus an audit-coverage gap drawn from a multi-level audit [@zharnikov-2026ar-orgschema-audit-six-level]. These are different kinds: one returns an eight-dimensional vector, the other a categorical audit verdict. The framework does not reconcile their raw outputs — there is no shared chain along which an eight-vector and a categorical audit could be combined. It reconciles the aligned claim's verdict — for instance an experiential-fit misfit on the unit interval in a merger-integration question — on which both instruments can be brought to a common axis. In a worked demonstration both endpoints are instrument-owned and their floors are *read live* from committed artifacts, while the claim values are illustrative inputs chosen to exercise the verdict regions (not field measurements): the measurement instrument resolves a magnitude of .0812 against an operator floor of .0103 read from a committed atlas, and the specification instrument resolves a transfer gap of .21 against a live combined coherence floor of .083 while abstaining on a .05 gap below it. Each floor has a single owner — a verification step (the *verify-contract gate*) recomputes it live from the committed artifact rather than trusting a hand-entered number — so the value an instrument declares cannot drift from the value it would compute.

***Mandated abstention below the coherence floor***

For the specification instrument, abstention below the coherence floor is not a convention the framework adopts but a consequence it inherits. A coverage-impossibility result establishes that below a threshold no specification audit can certify coverage [@zharnikov-2026-specification-impossibility-organizational-design-high]; a specification instrument that resolved a finding below its coherence floor would therefore be asserting a result its own audit provably cannot support. Abstention is forced, not chosen. A specification instrument shown to resolve correctly below its coherence floor — coverage demonstrably certified where the impossibility result says it cannot be — would refute the mandated-abstention claim and, with it, the cross-kind unification.

## Relation to Prior Art

***Neighboring methods, and what each cannot do***

The lattice's nearest neighbors live in separate communities with no bridging construct between them; that fragmentation is itself the argument that the lattice is a new object rather than a relabeling. The broad uncertainty-quantification literature catalogs how to *represent* and *propagate* uncertainty within a model, distinguishing aleatoric from epistemic components and surveying the methods that estimate each [@huellermeier-2021-aleatoric-epistemic-uncertainty; @gawlikowski-2023-survey-uncertainty-deep-neural; @mena-2021-survey-uncertainty-bayesian; @he-2026-survey-uncertainty-quantification-deep-learning], but it stops short of a cross-instrument calculus that returns a typed verdict over heterogeneous instruments' aligned outputs. Six specific neighbors bound the contribution. *Meta-analysis heterogeneity* quantifies the dispersion of commensurable effect sizes and, under a random-effects model, reports a pooled estimate with a heterogeneity-widened interval [@higgins-2002-quantifying-heterogeneity-meta-analysis]; it requires a common effect-size metric, so it cannot compose instruments of different kinds, and it emits a pooled mean by default even under high heterogeneity. *Opinion pooling* asks how to combine several probability distributions into one under axioms such as external Bayesianity [@genest-zidek-1986-combining-probability-distributions]; it presumes that pooling occurs, where the lattice makes pooling conditional on clearing the effective floor and otherwise declines. *Sensor fusion and belief combination* combine multiple sources into a single estimate and can identify and down-weight a faulted source [@dempster-1967-upper-lower-probabilities]; but they fuse to one value, which can mask conflict, they down-weight rather than exclude, and they read agreement as confidence. *Truth discovery and data fusion* resolve conflicts among many sources by inferring each source's reliability and recovering a single most-credible value [@li-2016-survey-truth-discovery], typically assuming a shared schema; the lattice instead aligns vocabularies as an explicit step-0 and, rather than recovering one truth, types the disagreement against each source's *declared, audited* floor. *Metrology uncertainty budgets* propagate Type A and Type B uncertainty within one measurement chain to a combined standard uncertainty [@jcgm-2008-gum-uncertainty-measurement]; the construct is single-instrument and single-output-space, with no cross-kind reconciliation and no abstention rule. *Conformal and selective prediction, and deep ensembles* deliver distribution-free coverage or abstention for one model under exchangeability and treat ensemble disagreement as uncertainty [@angelopoulos-2023-conformal-prediction-gentle; @lakshminarayanan-2017-deep-ensembles]; but they operate on a single model, return a binary abstain-or-predict rather than a typed verdict, and do not compose instruments of different kinds. Two further traditions are preconditions the framework absorbs rather than competitors: *measurement invariance testing* asks whether a construct is measured equivalently across groups before they are compared [@putnick-2016-measurement-invariance-conventions] — a binary precondition the alignment step-0 operationalizes before proceeding to a typed verdict — and *multi-annotator latent-class models* estimate per-rater error rates to recover a latent label [@dawid-1979-maximum-likelihood-observer-error], where the lattice substitutes a declared, audited floor and exclusion for an inferred reliability weight. Table 1 summarizes, for each neighbor, what it does, what it structurally cannot do, and the concrete input on which its verdict and the lattice's diverge.

**Table 1.** Relation of the reconciliation lattice to neighboring methods.

| Neighboring method | What it does | What it cannot do | Divergent verdict (lattice vs. method) |
|---|---|---|---|
| Meta-analysis heterogeneity [@higgins-2002-quantifying-heterogeneity-meta-analysis] | Quantifies dispersion of commensurable effect sizes; pools with a widened interval | Cross-kind composition; suppressing the pooled mean under high heterogeneity | High heterogeneity → pooled mean with wide CI; lattice → *contested*, no pooled mean |
| Opinion pooling [@genest-zidek-1986-combining-probability-distributions] | Combines distributions under aggregation axioms | Declining to pool; a typed verdict | Below-floor inputs → a pooled distribution; lattice → *jointly-unresolved* |
| Sensor fusion / belief combination [@dempster-1967-upper-lower-probabilities] | Combines sources into one estimate; down-weights a faulted source | Excluding (not down-weighting); refusing rescue | Two below-floor sources agree → confident fused value; lattice → *jointly-unresolved* |
| Truth discovery / data fusion [@li-2016-survey-truth-discovery] | Infers source reliability; recovers one most-credible value | Aligning divergent vocabularies; typing the disagreement | Conflict beyond floors → one inferred truth; lattice → *contested* over audited floors |
| Metrology uncertainty budget [@jcgm-2008-gum-uncertainty-measurement] | Propagates Type A/B uncertainty in one chain | Cross-kind reconciliation; an abstention rule | 8-vector vs. categorical audit → no shared chain to propagate; lattice → aligned-verdict reconciliation |

***Three differentiating lemmas***

Each neighbor is distinguished by a lemma that exhibits an input on which the prior method and the lattice return different verdicts. **Lemma 4a (non-pooling).** Where high heterogeneity makes a random-effects meta-analysis report a wide interval around a pooled mean, the lattice emits contested and no pooled mean, because the four verdict types are not a function of any pooled point estimate. **Lemma 4b (exclusion and no-rescue).** Sensor fusion combines all sources into one estimate and down-weights a faulted source; the lattice excludes an uncertifiable source at the self-certification gate and, by the no-rescue lemma, forbids agreement among below-floor sources from producing any verdict — a different output on the exact inputs where fusion yields a confident combined value. **Lemma 4c (cross-kind).** A metrology budget propagates uncertainty within one chain and selective prediction abstains for one model under exchangeability; neither composes instruments of different kinds with no shared output space. The lattice reconciles an eight-vector and a categorical audit on the aligned ordinal verdict — an object none of the four constructs. Exhibiting, for any of the three lemmas, that the named prior method already returns the lattice's verdict on the claimed-divergent input would collapse that lemma.

## A Smallest Sufficient Model

The lemmas are characterized on the smallest model that carries them: two instruments on a normalized verdict axis in the unit interval, each emitting either a resolve with a value and a self-floor or an abstain. The interval floors are the value plus or minus the floor width; the substrate dispersion is the absolute difference of the two values; the effective floor is the maximum of the dispersion and the two floor widths. The four lattice rules partition the plane of dispersion against consensus-versus-floor into the four verdicts. This model yields the verdict-region landscape and the no-rescue and coverage-impossibility diagrams; the raw eight-vector and specification-text mechanics, which add no new structure to the verdict logic, are deferred to the reference implementation. Locking the unit of analysis to a single aligned ordinal claim is deliberate: it keeps the contribution the verdict calculus, not any one instrument's internal scale.

## Worked Demonstrations

***Four verdicts on shipped cases***

The framework is not a proposal but a committed reference implementation, and its worked cases reproduce all four verdicts with both instrument kinds' floors read live from artifacts. On a merger-integration experiential-fit claim, two instruments that resolve with a dispersion of .02 inside a maximum self-floor of .03 and a consensus of .10 clearing the effective floor at a signal-to-noise of 3.33 return *corroborated*. Raising the dispersion to .55 against a self-floor of .05 — disagreement an order of magnitude beyond the floor — returns *contested* at a signal-to-noise of .86, escalated by construction rather than averaged. A single instrument resolving at signal-to-noise 8.0 while the other abstains returns *substrate-conditional* with a maximally split resolve/abstain entropy of one. Two cases return *jointly-unresolved* for distinct reasons: when both instruments abstain there is no resolver set, and — the no-rescue lemma made concrete — when both resolve and agree within their floors (dispersion .01) but on a consensus of .025 that fails the effective floor at signal-to-noise .50, the agreement is on noise. A further case exercises the alignment gate: two instruments that agree numerically (dispersion .02, signal-to-noise 3.33) but whose anchor terms are unaligned in the live graph are downgraded from corroborated to *contested*, the false-agreement risk surfaced rather than rewarded.

***An instrument outside the corpus***

To establish the cross-kind and cross-author claim on an instrument outside the author's corpus, one demonstration reconciles a corpus measurement instrument with a genuinely external third-party brand-equity lens whose anchor concept is not in the corpus terms graph. The external lens self-certifies and declares a typed mapping from its own pooled-scalar equity index to a corpus concept — a close match, the same conceptual territory but a divergent operationalization, validated on the corpus side and carried as an alignment risk rather than silently trusted. The reconciliation returns *corroborated* (dispersion .02 within a .03 floor, consensus .11 at signal-to-noise 3.67) with the close-match alignment risk flagged on the external endpoint. The mechanism that makes a cross-author boundary tractable is the same typed-mapping standard the alignment gate uses [@matentzoglu-2022-sssom]; the external instrument is admitted not because it shares the corpus vocabulary but because it can declare, and have validated, how its vocabulary maps. A reconciliation of public benchmark leaderboards of different kinds — with each instrument's floor computed from its documented test-set size — exercises the same cross-author mechanism on fully external data, and is reported next.

***A public-benchmark reconciliation***

The same lattice runs, unchanged, on instruments and data entirely outside the corpus. Three frontier language models are scored by three public benchmarks of different kinds that share no raw output space: MMLU, a broad-knowledge multiple-choice test [@hendrycks-2021-mmlu]; MATH, competition mathematics with free-response answers [@hendrycks-2021-math-dataset]; and HumanEval, execution-tested code synthesis [@chen-2021-evaluating-llms-code]. Accuracies are a pinned, dated snapshot of the models' reported scores [@dubey-2024-llama-3-herd]; each benchmark's noise floor is the binomial standard error of its accuracy, computed from its documented test-set size — a conservative sampling-noise floor that differs sharply by kind through size alone: MMLU (14,042 items) carries a floor near .003, MATH (5,000) near .005, but HumanEval (164) near .028, an order of magnitude larger. The shared claim "model A outperforms model B" is read on a common ordinal axis via a two-proportion test per benchmark, and the reconciliation runs through the identical `substrate_floor` lattice the corpus cases use.

The verdicts are instructive precisely because they refuse what the raw leaderboard asserts. For the two closest top models the lattice returns *jointly-unresolved*: their .1-point knowledge gap and 1.2-point code gap both fall within the respective floors, so the instruments are statistically silent on which is stronger. For a clear knowledge-and-math advantage the lattice returns *corroborated* (both kinds resolve in agreement, signal-to-noise 2.71). Most tellingly, the widely-cited code ranking — one model leading another by one to three points on HumanEval — is *not* certified: those gaps fall below HumanEval's .028 floor, so the code instrument abstains, and the claim resolves only on the math axis, returning *substrate-conditional* (it holds under the mathematics apparatus, not the code one). The nested-floor discipline, applied to recognizable public data, declines to read a few-point gap on a 164-item benchmark as a finding — exactly the abstention a leaderboard's point ranking omits.

## Monte-Carlo Verdict Regions

***The verdict landscape***

The empirical content of the differentiation claim is the share of cases on which the lattice and each prior method disagree. A seeded Monte-Carlo sweep of the two-instrument model draws verdict values under each of three priors — uniform on the unit interval (the uninformative headline), beta(2,5) (claims skewed weak), and a truncated normal centered at the axis midpoint — with floor widths uniform up to half the axis, and classifies each draw (200,000 draws per prior, seed 20260624). Under the uniform prior the verdict regions are corroborated .120, contested .230, substrate-conditional .250, and jointly-unresolved .400: the lattice abstains or qualifies far more often than it corroborates, the intended conservatism. The regions shift with the prior — under beta(2,5) corroboration falls to .032 as weaker claims more often fail their floors — but the differentiation results below hold in order across all three priors.

***Where the verdicts diverge***

The divergence fractions quantify the three lemmas, each reported with a 95% nonparametric bootstrap confidence interval (1000 resamples) under the uniform prior, followed by its range across the three priors. Against a random-effects pool, which would assert a pooled point estimate on every draw, the lattice withholds that estimate on .880 of cases (95% CI [.878, .881]) — every draw it does not return corroborated — and on .87 to .97 of cases across the three priors (Lemma 4a). The no-rescue region — both instruments resolving and agreeing within their floors yet on a consensus below the effective floor, where sensor fusion would emit a confident combined value — accounts for .214 of cases ([.212, .216]; .18 to .40 across priors), the share on which the lattice returns jointly-unresolved where fusion returns a number (Lemma 4b). And .480 of cases carry a typed verdict — contested or substrate-conditional — that a binary abstain-or-predict rule cannot represent (.480, [.477, .482]; .34 to .48 across priors; Lemma 4c). These are not small residuals at the boundary: under every prior, on a third to nearly all of the space depending on the lemma, the lattice returns something the named neighbor, by construction, does not.

## Robustness

Three checks probe the verdicts away from the worked points. *Boundary sensitivity*: varying the resolve and marginal multipliers and the alignment predicate (exact versus close) confirms the verdict types are stable away from the region boundaries and that the alignment risk correctly demotes confidence near them. *Correlated-error stress*: relaxing the conditional-independence assumption by injecting shared systematic error below the declared floors confirms that the no-rescue lemma still bounds rescue from above — it is a statement about each endpoint's own floor, not about independence — while corroborated verdicts are correctly downgraded, since correlated error makes the dispersion under-state true disagreement. *Cross-method divergence audit*: the Monte-Carlo grid quantifies the divergence fractions reported above under three priors with bootstrap confidence intervals, giving the magnitude of the differentiation and its stability across the value distribution rather than only its existence.

## Discussion

***What the framework refuses to do***

The framework's value is most visible in what it declines. It declines to pool incommensurable instruments into a single number; it declines to let an uncertified instrument vote; it declines to read agreement among noisy instruments as evidence; and it declines, below a coherence floor, to resolve at all. The capacity to return *contested* or *substrate-conditional* rather than a forced point estimate is, in high-stakes settings, itself a safety property: the documented need for machine-learning systems to communicate honestly when they should abstain — to say *I don't know* rather than predict overconfidently — is exactly what a typed verdict supplies at the ensemble level [@kompa-2021-second-opinion-medical-ml]. Two objections are worth meeting directly. The first is that the lattice is heterogeneity reporting with new labels. The three lemmas answer it concretely: each exhibits an input on which the named prior method returns a different verdict — no pooled mean under contested, jointly-unresolved against confident fusion, a cross-kind reconciliation with no shared chain — and the Monte-Carlo fractions measure how often. The contribution is the typed-verdict object and the mandated abstention, not the heterogeneity statistic, which the framework reports rather than replaces. The second objection is that ensemble disagreement already captures the intuition that disagreeing instruments warrant low confidence. It does not capture the cases that matter: it reads confident agreement among uncertified black boxes as evidence — the floorless-black-box non-application — and it has no typed verdict, no self-certification exclusion, and no cross-kind axis. The self-certification gate and the no-rescue lemma are precisely the situations in which naïve disagreement scoring fails.

***Auditable pattern detection across heterogeneous sources***

The practical contribution is a reproducible primitive for pattern detection when the evidence comes from sources of different kinds. A pipeline that mines a pattern from heterogeneous instruments — embeddings, model audits, benchmark scores, human coding — can now attach to each reported pattern a typed verdict and a provenance trail, rather than a single pooled number whose reconciliation step is invisible. This is the step the reproducibility literature identifies as a common point of failure: an unaudited methodological choice that silently converts insufficient evidence into a confident finding [@munafo-2017-manifesto-reproducible-science; @national-academies-2019-reproducibility-replicability]. The public-benchmark reconciliation makes the gain concrete: faced with a few-point gap on a small benchmark, the lattice reports *substrate-conditional* or *jointly-unresolved* and names the apparatus, where a leaderboard reports a rank. For an automated science pipeline the value is the same — a pattern is published with the conditions under which it holds, the instruments that corroborate it, and an explicit refusal where the evidence is below the floor. Reproducibility is delivered not by a new statistic but by making the reconciliation auditable.

***Limitations***

Three limitations bound the claim. The demonstrations span corpus-internal instruments, one external vendor lens, and a public-benchmark reconciliation, but the latter uses reported accuracies under each model's chosen evaluation conditions and a sampling-only floor; a fully controlled study that re-runs each instrument under fixed conditions and adds prompt-sensitivity to the floor would tighten the external evidence, and remains the most valuable next addition. The common verdict axis assumes a defensible alignment exists; where alignment is genuinely contested the framework surfaces the risk but cannot itself adjudicate the meaning dispute, which is handed to the upstream negotiator [@zharnikov-2026at-negotiating-vocabularies-link] and, failing that, to human judgment. And floors are only as honest as their source artifacts; an instrument with neither a live floor reference nor a certifiable declaration is inadmissible, which narrows applicability to instruments that have done the metrology work — a deliberate scope cost, not a defect. Within these bounds the rule is general: it asks of any instrument only that it align, expose a floor, and certify itself, and in return it composes that instrument with any other on a common verdict without reintroducing a hidden oracle.

## Conclusion

Honest abstention need not stop at the edge of a single instrument. Under one nested-floor rule — clear the outermost floor among operator, artifact, and substrate — the discipline that lets one measurement instrument withhold a reading lets an ensemble of heterogeneous instruments withhold a finding, and a no-rescue lemma guarantees that agreement among noisy instruments cannot manufacture one. Reconciliation returns a typed verdict rather than a forced estimate, gated by alignment and self-certification, and the rule composes instruments of different kinds — a measurement vector and a specification audit — on an aligned verdict where their raw outputs cannot be combined. The framework is shipped and reproducible: worked cases reproduce all four verdicts, and a seeded Monte-Carlo characterization marks where the lattice returns what meta-analysis, sensor fusion, metrology, and selective prediction do not. The result is a way for an agent to compose what it knows from many sources and report, auditably, what is resolved, what is contested, and what cannot be told.

## Appendix A. Proof of Proposition 1 and the Verdict Algorithm

***Proof of Proposition 1 (no rescue)***

Let instruments $1, \dots, n$ resolve a shared claim on a normalized verdict axis, each with verdict value $v_i$ and combined noise floor $f_i > 0$. Let $D = \max_{i,j} |v_i - v_j|$ be the substrate dispersion, $c$ the consensus signal, and $\mathrm{effective\_floor} = \max(D, f_1, \dots, f_n)$. By the definition of the maximum, $\mathrm{effective\_floor} \ge f_k$ for every $k$. A finding survives iff $|c| > \mathrm{effective\_floor}$. Suppose some endpoint $k$ has $|c| \le f_k$. Then $|c| \le f_k \le \mathrm{effective\_floor}$, so the survival inequality fails. Hence whenever any single endpoint's signal lies at or below its own floor, the consensus cannot clear the effective floor, irrespective of the magnitude of cross-instrument agreement (which can only shrink $D$, never $\max_k f_k$, and therefore never lowers $\mathrm{effective\_floor}$ below $\max_k f_k$). $\square$

The corollary used in the main text is immediate: agreement reduces $D$ but leaves $\max_k f_k$ untouched, so for two endpoints that agree exactly ($D = 0$) the effective floor is still $\max(f_1, f_2)$, and a consensus below that floor yields *jointly-unresolved* (agreement on noise), not *corroborated*.

***Algorithm 1 (LatticeVerdict)***

```
input : verdicts V = {(v_i, f_i, resolve_i, align_i, audit_i)}, thresholds k_resolve, k_marginal
output: a typed verdict and a resolve/abstain entropy

1  drop every v_i with audit_i = fail            # self-certification gate
2  if any pair (i, j) of resolvers is unaligned in the graph:
3      mark alignment_risk; an unaligned pair caps a would-be corroborated verdict at contested
4  R <- { i : resolve_i and v_i > k_marginal * f_i }   # the resolving set
5  if |R| = 0:            return jointly-unresolved
6  if |R| = 1:
7      sn <- |v_R| / f_R
8      return substrate-conditional if sn >= k_resolve else jointly-unresolved
9  D   <- max pairwise |v_i - v_j| over R
10 fmax<- max f_i over R ;  eff <- max(D, fmax) ;  c <- mean(v_i over R) ;  sn <- |c| / eff
11 if D > fmax:           return contested            # disagree beyond floors
12 if sn >= k_resolve:    return corroborated unless alignment_risk else contested
13 return jointly-unresolved                          # agreement on noise (Proposition 1)
```

The reference implementation `code/substrate_floor.py` realizes Algorithm 1 line for line; the vectorized companion `code/verdict_regions_mc.py` reproduces it for the Monte-Carlo characterization.
